## Marquee Files

All flyers must be in a landscape 1376x768 size.

All flyer files must be a PNG or APNG file with the .png file extension.

All files must be named flyer_#.png, where # is a one- or two-digit number ranging from 1 to 50.

Files that do not match the flyer_#.png naming convention will be ignored.

Numbers can be skipped.

Flyers can be deleted and added to the folder as needed.

No other files need to be edited; changes are automatically detected.

It could take a little bit of time before all changes update.
